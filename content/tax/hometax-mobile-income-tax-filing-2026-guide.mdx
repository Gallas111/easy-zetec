## 5월 되니까 가슴이 답답하시죠?

"종합소득세 신고해야 하는데 홈택스 들어가면 어디부터 눌러야 할지 모르겠어..."
작년에 저도 그랬어요. PC 홈택스 들어가면 메뉴가 열 개 넘게 보여서 바로 포기했거든요.

그래서 이번엔 **모바일 앱**으로 해봤어요. 진짜 30분이면 끝나요.
공동인증서도 없이 카카오 간편인증으로 로그인했고, 카드 내역도 자동으로 불러왔어요.

이 글에서는 2026년 4월 기준 홈택스 앱 최신 화면 순서대로 5단계를 정리했어요.
프리랜서·부업 직장인 모두 그대로 따라 하실 수 있습니다.

![홈택스 모바일 세금 신고](/images/posts/hometax-mobile-income-tax-filing-2026-guide-thumb.webp)

## 신고 전 준비물

- **홈택스 앱 최신 버전** (App Store/Play Store에서 "홈택스" 검색)
- **간편인증 수단**: 카카오톡·네이버·PASS·토스 중 하나
- **환급받을 계좌번호**: 본인 명의 계좌
- **필요경비 자료** (프리랜서·사업자만): 카드 사용 내역, 영수증 사진

약 10분이면 준비 끝이에요. 공동인증서 없어도 됩니다.

## Step 1 — 홈택스 앱 로그인

앱을 열면 홈 화면 상단에 "로그인" 버튼이 있어요. 탭 하면 인증 수단 선택 화면이 나옵니다.

### 간편인증 사용법

"간편인증" 탭 → 수단 선택(카카오톡이 가장 빠름) → 휴대폰으로 인증 요청 전송 → 카카오톡에서 터치 한 번 → 완료.

공동인증서 쓰던 시절엔 5분 걸리던 게 20초예요. 2026년 4월 현재 간편인증 안정성도 99% 수준이라 공동인증서는 거의 필요 없어졌어요.

![세금 계산기 스마트폰](/images/posts/hometax-mobile-income-tax-filing-2026-guide-body-1.webp)

## Step 2 — 종합소득세 신고 메뉴 진입

로그인하면 메인 화면에 **"국세 신고"** 카테고리가 보여요. 여기서 "종합소득세"를 탭하세요.

종합소득세 화면 안에는 세 가지 옵션이 있어요.

| 옵션 | 대상 |
|------|------|
| 정기 신고 | 5월 1일~5월 31일 일반 신고 |
| 기한 후 신고 | 6월 1일 이후 늦은 신고 |
| 수정 신고 | 이미 신고한 내용 바로잡기 |

5월 중이면 "정기 신고"를 선택합니다. 앱이 자동으로 본인 소득 유형(근로·사업·기타)을 분석해서 맞는 신고서 양식을 띄워줘요.

## Step 3 — 소득 내역 확인 및 수정

앱이 본인 소득을 자동으로 불러옵니다. 확인만 하면 되는 경우가 대부분이에요.

### 자동 불러오기 항목

- 근로소득: 회사 연말정산 데이터
- 사업소득: 세금계산서·현금영수증 합산
- 기타소득: 강의료·원고료·인세
- 배당·이자소득: 금융기관 통보 내역

### 직접 입력이 필요한 경우

- 해외 소득 (유튜브·앱스토어 수입 등)
- 3.3% 원천징수 누락된 프리랜서 수입
- 소액 현금 거래

입력 필드는 한국어 + 물음표 아이콘 도움말이 붙어 있어서 헷갈릴 때 바로 설명이 나와요.

## Step 4 — 필요경비 입력

사업소득자·프리랜서가 가장 많이 놓치는 부분이에요. 필요경비는 **소득에서 차감되는 비용**으로, 정확히 입력할수록 세금이 줄어요.

### 단순경비율 vs 기준경비율

소득 규모에 따라 두 방식 중 하나를 선택합니다.

| 구분 | 대상 | 특징 |
|------|------|------|
| 단순경비율 | 매출 2,400만원 미만 | 업종별 정해진 비율 자동 적용, 영수증 불필요 |
| 기준경비율 | 매출 2,400만원 이상 | 주요 경비만 영수증 증빙, 나머지 비율 적용 |
| 장부기장 | 매출 7,500만원 이상 | 전 경비 증빙 필수, 세액 가장 줄어듦 |

첫 신고자는 단순경비율이 훨씬 편해요. 매출이 2,400만원 미만이면 고민할 필요 없이 선택하세요.

### 영수증 첨부

기준경비율이나 장부기장 대상이면 카드 사용 내역이 앱에 자동 연동돼 있어요. "경비 항목별 카드 내역 불러오기" 탭 하면 식비·교통비·통신비 등 카테고리로 정리됩니다.

![사무실 서류 노트북](/images/posts/hometax-mobile-income-tax-filing-2026-guide-body-2.webp)

## Step 5 — 세액 확인 후 제출

모든 입력이 끝나면 **예상 세액**이 화면에 나와요.

- 플러스 금액: 추가 납부 (분납 가능, 2회 차는 8월 31일까지)
- 마이너스 금액: 환급 (6월 말 경 지정 계좌로 입금)

"신고서 미리보기" 버튼으로 PDF 내용을 확인한 뒤 "제출" 버튼을 누르면 끝. **접수증 PDF를 꼭 저장**하세요. 나중에 은행 대출 받을 때 소득증빙 자료로 쓰입니다.

## 직장인 부업 케이스

회사 월급 + 블로그 애드센스 + 유튜브 수익이 있는 경우예요.

1. 근로소득(월급)은 앱에 자동 입력돼 있음
2. 블로그 애드센스·유튜브 수익은 **기타소득** 탭에서 직접 입력
3. 기타소득 연 300만원 이하면 분리과세 선택 가능 (단일세율 22%)
4. 300만원 초과면 종합과세로 합산 신고

애드센스는 원천징수가 없으니 꼭 자진 신고해야 해요. 국세청이 구글 서버와 데이터 공유 중이라 누락하면 3년 내 추적 통지가 옵니다.

## 신고 후 꼭 체크할 것

- **접수번호 확인**: 제출 완료 화면의 접수번호 스크린샷
- **환급 계좌 재확인**: 계좌 오류 시 환급 지연
- **수정 기간 활용**: 5월 31일 전까지 몇 번이든 수정 가능

## 자주 하는 실수

- **카카오페이 등 간편결제 수익 누락** — 자영업자는 이 부분 꼭 챙기세요
- **해외 수익 원화 환산 오류** — 국세청 고시 환율 사용 필수
- **부업 소득 분리과세만 신청** — 금액에 따라 종합과세가 유리할 수 있음
- **신고 기한 지나서 당황** — 기한 후 신고도 가능하고 환급도 되지만 가산세는 붙어요

## 모바일 신고 자주 묻는 추가 질문

홈택스 앱 사용 중 자주 묻는 질문을 정리했어요.

### 앱이 자꾸 꺼지면 어떻게 하나요

5월 신고 시즌에 사용자가 몰려 앱이 불안정할 수 있어요. 5월 첫 주 또는 마지막 주를 피하고 평일 오전이나 늦은 밤 시간대를 활용하세요. 와이파이 환경에서 신고 권장이에요.

### 신고 도중 종료되면 데이터가 보존되나요

앱에서 자동 임시 저장 기능이 있어 신고 도중 종료되어도 입력 내용이 보존돼요. 다시 로그인하면 이어서 작성 가능해요. 단, 너무 오래 미루면 세션 만료될 수 있어요.

### PC 홈택스와 차이가 있나요

기본 기능은 동일하지만 일부 복잡한 항목은 PC에서만 가능해요. 간편장부 대상자는 PC 작성 권장이에요. 단순 신고는 모바일이 훨씬 편해요.

### 가족 명의로 신고 가능한가요

본인 명의 인증서로만 본인 신고 가능해요. 가족 대신 신고하려면 가족 명의 인증 수단이 필요해요. 부모님 신고 대행 시 위임장 작성 가능한 경우도 있어요.

### 신고 후 신고서 다운로드 방법

신고 완료 후 신고서 PDF를 즉시 다운로드 가능해요. 앱 마이페이지에서 언제든 재다운로드 가능해요. 5년간 자료 보관 의무가 있으니 별도 저장 권장이에요.

## 모바일 신고 시 영수증 처리 팁

영수증 입력은 가장 헷갈리는 부분이에요. 체계적으로 정리하면 효율적이에요.

### 카드 자동 연동 활용

홈택스에 사업용 카드를 등록하면 매월 사용 내역이 자동 수집돼요. 5월 신고 시 카드 내역이 자동 불러오기 되어 별도 입력 불필요해요. 사업용 카드와 개인용 카드를 분리하는 게 중요해요.

### 현금영수증 확인

홈택스에서 현금영수증 발행 내역을 조회할 수 있어요. 누락된 영수증이 없는지 점검하세요. 사업자등록번호로 발행받은 현금영수증은 자동 경비 처리돼요.

### 영수증 사진 첨부

종이 영수증은 모바일 앱에서 사진 촬영해서 첨부 가능해요. 영수증 발행일, 금액, 거래처가 명확히 보이도록 촬영하세요. 흐릿한 사진은 인정 안 될 수 있어요.

### 카테고리별 정리

식비, 교통비, 통신비, 사무용품비 등 카테고리별로 정리해두면 신고가 빨라져요. 월별 정리도 권장이에요.

### 영수증 보관 의무

신고 후에도 영수증을 5년간 보관해야 해요. 세무 조사 시 증빙 자료로 활용돼요. 종이 영수증은 디지털 스캔으로 보관 가능해요.

## 신고 후 환급 받기까지 일정

신고 완료 후 환급금이 입금되기까지의 일정을 정리했어요.

### 정기 신고 후 환급 일정

5월 31일까지 신고 완료한 경우 6월 말에서 7월 초에 환급금이 입금돼요. 일반적으로 신고 완료 후 약 한 달 안에 처리돼요.

### 기한 후 신고 환급

6월 1일 이후 기한 후 신고한 경우 처리 기간이 길어져요. 약 2~4주 추가로 소요될 수 있어요.

### 환급 진행 확인

홈택스 앱 또는 PC 홈택스에서 환급 진행 상황을 실시간 조회 가능해요. 처리 중, 입금 예정 등 단계별 상태가 표시돼요.

### 환급 계좌 변경

신고 후 환급 계좌를 변경하려면 홈택스에서 별도 신청해야 해요. 입금 직전까지는 변경 가능해요.

## 모바일 앱 활용 5가지 시나리오

본인 상황에 맞는 신고 방법을 시나리오로 정리했어요.

### 시나리오 1: 단순 근로자 부수입 없음

회사 연말정산으로 대부분 끝났고 부수입이 없는 경우예요. 5월 종합소득세 신고 의무 없어요. 단, 의료비나 기부금 등 누락 공제가 있으면 경정청구로 추가 환급 가능해요.

### 시나리오 2: 근로자 부수입 있는 경우

본업 월급 외에 강의료, 인세, 임대 소득 등 부수입이 있는 경우 5월 종소세 신고 필수예요. 모바일 앱에서 자동 불러오기 기능 활용하면 30분 안에 완료돼요. 부수입 300만원 이하 분리과세 선택 가능해요.

### 시나리오 3: 프리랜서 연 수입 3,000만원 미만

단순경비율 적용 대상으로 영수증 없이도 자동 경비 처리돼요. 모바일 앱 신고가 가장 효율적이에요. 약 30분 안에 신고 완료 가능해요.

### 시나리오 4: 프리랜서 연 수입 5,000만원 이상

기준경비율 적용 대상으로 영수증 정리 필요해요. 모바일 앱에서도 카드 사용 내역 자동 불러오기 가능하지만 PC에서 작업이 더 편할 수 있어요. 첫 신고는 세무사 자문 권장이에요.

### 시나리오 5: 자영업자 사업자 등록

사업자 등록한 자영업자는 부가세 신고와 종소세 신고 모두 의무예요. 부가세는 1월과 7월, 종소세는 5월에 각각 신고해요. 세무사 의뢰가 일반적이에요.

## 신고 시 주의해야 할 함정 정리

신고 단계에서 자주 빠지는 함정을 정리했어요.

### 자동 불러오기 신뢰

자동 불러오기 자료가 항상 정확한 건 아니에요. 누락 항목이 있을 수 있으니 직접 확인하세요. 특히 해외 소득이나 일부 부수입은 자동 조회 안 될 수 있어요.

### 인적공제 검증

부양가족 정보가 자동 입력되지만 누락이나 오류가 있을 수 있어요. 부양가족 본인 소득이 100만원 이상이면 부양가족 공제 안 돼요.

### 카드 사용액 점검

카드 사용 내역도 자동 불러오기 되지만 사업용과 개인용 구분이 부정확할 수 있어요. 본인 직접 검토 후 수정 필수예요.

### 환급 계좌 정확성

환급 받을 계좌번호를 두 번 확인하세요. 한 번 잘못 입력하면 환급 지연되거나 환수 처리될 수 있어요.

### 5월 마감일 임박 시 시스템 폭주

5월 마지막 주는 시스템이 폭주해서 앱이 불안정해요. 가능하면 5월 중순까지 신고 완료 권장이에요.

## 모바일 앱 보안 점검

모바일 신고 시 보안 관리도 중요해요.

### 인증 수단 보안

본인 명의 인증 수단만 사용하세요. 가족 명의 인증서나 간편인증으로 신고 시 부정 신고로 분류될 수 있어요. 인증 정보를 다른 사람에게 알려주지 마세요.

### 와이파이 보안

공용 와이파이에서 신고는 피하세요. 보안 위험이 있어요. 본인 집 와이파이나 모바일 데이터 사용 권장이에요.

### 앱 최신 버전

홈택스 앱을 최신 버전으로 유지하세요. 구버전은 보안 취약점이 있을 수 있어요. 정기 업데이트 확인 권장이에요.

### 비밀번호 관리

간편인증 비밀번호와 인증서 비밀번호를 정기적으로 변경하세요. 동일 비밀번호를 여러 곳에서 사용하면 위험해요.

### 로그아웃 습관

신고 완료 후 반드시 로그아웃하세요. 자동 로그인 설정은 보안상 위험할 수 있어요.

### 다른 사람과 함께 사용 금지

본인 휴대폰을 다른 사람과 공유 사용 시 위험해요. 본인 명의 휴대폰에서만 신고 권장이에요. 잠금 화면 비밀번호 설정도 중요해요.
