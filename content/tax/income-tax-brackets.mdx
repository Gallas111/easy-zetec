"세금을 얼마나 내야 하는 거지?" 소득이 생기면 누구나 한 번쯤 궁금해하는 질문입니다. 특히 직장인, 프리랜서, 사업자 모두에게 해당되는 **종합소득세**는 세율 구간을 이해하는 것만으로도 절세의 첫걸음이 됩니다.

오늘은 2026년 기준 종합소득세 세율 구간을 정리하고, 실제로 세금이 어떻게 계산되는지 **쉽고 명확하게** 알려드리겠습니다.

## 종합소득세란?

종합소득세는 개인이 1년간 벌어들인 **모든 소득을 합산**하여 부과하는 세금입니다.

합산 대상이 되는 소득은 다음과 같습니다:

- **근로소득** (직장 급여)
- **사업소득** (자영업, 프리랜서)
- **이자소득** (예금 이자)
- **배당소득** (주식 배당금)
- **연금소득**
- **기타소득** (강연료, 원고료 등)

> 직장인은 연말정산으로 끝나는 경우가 많지만, **부수입이 있다면 5월에 종합소득세 신고**를 별도로 해야 합니다!

## 2026년 종합소득세 세율 구간표

한국의 소득세는 **누진세** 구조입니다. 소득이 높을수록 더 높은 세율이 적용되는 방식이에요.

| 과세표준 | 세율 | 누진공제액 |
|---------|------|----------|
| 1,400만원 이하 | **6%** | - |
| 1,400만원 초과 ~ 5,000만원 이하 | **15%** | 126만원 |
| 5,000만원 초과 ~ 8,800만원 이하 | **24%** | 576만원 |
| 8,800만원 초과 ~ 1억 5,000만원 이하 | **35%** | 1,544만원 |
| 1억 5,000만원 초과 ~ 3억원 이하 | **38%** | 1,994만원 |
| 3억원 초과 ~ 5억원 이하 | **40%** | 2,594만원 |
| 5억원 초과 ~ 10억원 이하 | **42%** | 3,594만원 |
| 10억원 초과 | **45%** | 6,594만원 |

> 여기에 **지방소득세 10%** 가 추가됩니다. 즉, 실제 최고세율은 45% + 4.5% = **49.5%** 입니다.

## 누진세, 제대로 이해하기

많은 분들이 세율 구간을 오해합니다. "내 소득이 5,000만원을 넘으면 **전부** 24% 세율이 적용되는 거 아닌가?"라고 생각하시죠.

**아닙니다!** 누진세는 구간별로 나누어 계산합니다.

### 잘못된 이해 vs 올바른 이해

| | 잘못된 이해 | 올바른 이해 |
|--|-----------|-----------|
| 과세표준 6,000만원 | 6,000만원 x 24% = 1,440만원 | 구간별로 나누어 계산 |
| 실제 세금 | 1,440만원 (X) | **864만원** (O) |

구간별로 계산하면 이렇습니다:

- 1,400만원 x 6% = **84만원**
- 3,600만원 (1,400~5,000만원) x 15% = **540만원**
- 1,000만원 (5,000~6,000만원) x 24% = **240만원**
- 합계: **864만원**

> 누진공제를 활용하면 더 간단합니다: 6,000만원 x 24% - 576만원 = **864만원** (같은 결과!)

## 실전 세금 계산 예시

### 예시 1: 연봉 4,000만원 직장인

| 항목 | 금액 |
|------|------|
| 총급여 | 4,000만원 |
| 근로소득공제 | -1,075만원 |
| 근로소득금액 | 2,925만원 |
| 인적공제 (본인) | -150만원 |
| 국민연금 공제 | -약 180만원 |
| 기타 소득공제 | -약 200만원 |
| **과세표준** | **약 2,395만원** |

세금 계산: 2,395만원 x 15% - 126만원 = **약 233만원**

여기서 세액공제(근로소득세액공제, 연금저축 등)를 빼면 **실제 납부액은 더 줄어듭니다**.

### 예시 2: 프리랜서 연수입 8,000만원

| 항목 | 금액 |
|------|------|
| 총수입 | 8,000만원 |
| 필요경비 (단순경비율 적용 시) | -약 5,000만원 |
| 소득금액 | 3,000만원 |
| 인적공제 + 기타공제 | -약 350만원 |
| **과세표준** | **약 2,650만원** |

세금 계산: 2,650만원 x 15% - 126만원 = **약 271.5만원**

> 프리랜서는 경비 처리에 따라 세금이 크게 달라집니다. **장부 기장**을 하면 실제 경비를 인정받아 세금을 더 줄일 수 있어요.

## 실효세율이란?

세율 구간만 보면 세금이 엄청 많아 보이지만, 실제로 내는 세금의 비율(**실효세율**)은 훨씬 낮습니다.

| 과세표준 | 최고 적용 세율 | 실효세율 |
|---------|-------------|---------|
| 2,000만원 | 15% | 약 10.8% |
| 4,000만원 | 15% | 약 12.9% |
| 7,000만원 | 24% | 약 16.8% |
| 1억원 | 35% | 약 21.1% |

실효세율은 각종 공제를 반영하기 전의 수치이므로, 실제 체감 세율은 이보다 더 낮을 수 있습니다.

## 종합소득세 줄이는 5가지 절세 팁

### 1. 연금저축 + IRP 세액공제 활용
연간 최대 900만원 납입 시 **최대 148.5만원** 세액공제를 받을 수 있습니다.

### 2. ISA 계좌 활용
투자 수익에 대해 **200~400만원 비과세** 혜택을 누릴 수 있습니다.

### 3. 소득 분산 전략
부부 공동사업의 경우 소득을 나누면 낮은 세율 구간이 적용되어 전체 세금이 줄어듭니다.

### 4. 필요경비 꼼꼼히 챙기기 (사업자·프리랜서)
- 사무실 임차료, 교통비, 통신비, 교육비 등
- **간편장부** 또는 **복식부기** 기장 시 추가 공제 가능

### 5. 세액공제 항목 최대한 활용
- 자녀 세액공제, 의료비, 교육비, 기부금
- 월세 세액공제 (총급여 8,000만원 이하 무주택자)

## 흔한 오해 바로잡기

- **"연봉이 오르면 오히려 손해다?"** → 세후 소득은 항상 증가합니다. 높은 세율은 추가 소득 부분에만 적용되기 때문입니다.
- **"세금은 연봉 기준으로 매긴다?"** → 아닙니다. 각종 공제를 뺀 **과세표준** 기준입니다. 연봉과 과세표준은 큰 차이가 있습니다.
- **"프리랜서는 세금을 더 많이 낸다?"** → 반드시 그렇지 않습니다. 경비 처리를 잘하면 직장인보다 유리할 수 있습니다.
