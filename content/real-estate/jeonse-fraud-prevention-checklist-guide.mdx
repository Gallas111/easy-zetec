"전세 사기 뉴스를 볼 때마다 무서워요. 나도 당할 수 있을까?"

솔직히 몇 년 전까지만 해도 전세 사기가 이렇게 대규모로 터질 줄 몰랐어요.

하지만 체크리스트만 잘 확인하면 대부분의 전세 사기를 사전에 예방할 수 있어요.

오늘은 계약 전 반드시 확인해야 할 7가지를 정리할게요.

## 체크리스트 1: 등기부등본 꼼꼼히 확인

등기부등본은 전세 계약의 가장 기본이에요.

### 확인할 항목

**갑구 (소유권)**
- 실제 집주인이 맞는지 확인
- 가압류, 가처분, 경매 기입이 있는지 확인
- 소유자가 여러 명이면 전원의 동의가 필요

**을구 (근저당권)**
- 근저당 설정 금액 확인
- 근저당 + 전세금이 집값의 70%를 넘지 않는지 확인

인터넷등기소(iros.go.kr)에서 700원에 발급받을 수 있어요. **계약 당일 다시 발급**받아서 확인하는 게 안전해요. 계약 전과 당일 사이에 근저당이 추가될 수 있거든요.

![부동산 등기부등본 확인 이미지](https://images.unsplash.com/photo-1560518883-ce09059eeffa?w=800&q=80)

## 체크리스트 2: 깡통전세 여부 확인

깡통전세는 전세 사기의 핵심 위험이에요.

### 깡통전세 확인 공식

**집값 x 0.7 > 근저당 + 전세금** → 안전
**집값 x 0.7 < 근저당 + 전세금** → 위험 (깡통전세)

예시:
- 집값: 3억원
- 근저당: 1억원
- 전세금: 1.5억원
- 3억 x 0.7 = 2.1억 > 1억 + 1.5억 = 2.5억 → 위험

### 시세 확인 방법
- 국토교통부 실거래가 공개시스템
- 네이버 부동산 시세
- KB부동산 시세

여러 곳에서 확인하고, 가장 낮은 시세를 기준으로 계산하세요.

## 체크리스트 3: 전세보증보험 가입 가능 여부

전세보증보험은 집주인이 보증금을 못 돌려줄 때 보험으로 돌려받을 수 있는 제도예요.

### 보증 기관
- HUG (주택도시보증공사): 가장 많이 이용
- SGI (서울보증보험): HUG 거절 시 대안
- HF (한국주택금융공사)

### 가입 조건
- 전세금이 시세의 일정 비율 이내
- 집주인이 세금 체납이 없을 것
- 전입신고 + 확정일자 완료

**계약 전에 가입 가능 여부를 먼저 확인하세요.** 가입이 안 되는 물건은 그만큼 위험도가 높다는 뜻이에요.

## 체크리스트 4: 집주인 신원 확인

### 대리인 계약 주의
집주인 본인이 아닌 대리인과 계약할 때는 특히 조심하세요.

확인할 것:
- 위임장 (인감도장 날인)
- 인감증명서 (최근 3개월 이내)
- 집주인 본인에게 직접 전화 확인

### 세금 체납 확인
집주인의 국세 체납 여부를 확인할 수 있어요. 집주인에게 납세증명서를 요청하거나, [홈택스](/blog/income-tax-filing-method-hometax-guide)에서 동의를 받아 확인할 수 있어요.

## 체크리스트 5: 전입신고와 확정일자 즉시 처리

![전입신고 확정일자 설명 이미지](https://images.unsplash.com/photo-1450101499163-c8848c66ca85?w=800&q=80)

이사한 당일에 바로 처리해야 해요.

### 전입신고
- 주민센터 방문 또는 정부24 온라인 신고
- 대항력 발생 (다음 날 0시부터)
- 경매 시 보증금 보호의 기본 조건

### 확정일자
- 주민센터에서 전입신고할 때 함께 신청 (600원)
- 우선변제권 확보
- 경매 시 배당 순위에서 유리한 위치

이 두 가지를 놓치면 보증금 보호를 받을 수 없어요.

## 체크리스트 6: 임대차 계약서 필수 조항

계약서에 이 항목들이 있는지 확인하세요:

- 보증금 반환 날짜와 방법
- 근저당 설정 제한 특약 ("계약 후 추가 근저당 설정 금지")
- 전세보증보험 가입 협조 조항
- 집주인 세금 체납 시 즉시 해지 조항
- 수리 비용 부담 주체 명시

특약은 **구두 합의가 아닌 계약서에 기재**해야 법적 효력이 있어요.

## 체크리스트 7: 전세 사기 예방 앱 활용

정부와 민간에서 제공하는 확인 도구를 활용하세요.

- **안심전세 앱 (HUG)**: 전세가율, 보증 가능 여부 확인
- **전세피해지원센터**: 전세 사기 의심 시 상담
- **국토교통부 실거래가**: 주변 시세 확인
- **등기부등본 알림 서비스**: 등기 변동 시 알림

[부동산 전망](/blog/real-estate-market-outlook-2026)을 함께 확인하면 시세 변동 위험도 파악할 수 있어요.

## 전세 사기 유형별 대응

### 1. 다주택자 전세 사기
한 사람이 여러 채를 사고 전세금으로 돌려막기 → **집주인 소유 부동산 수 확인**

### 2. 신축 빌라 사기
시세보다 높은 전세금 → **실거래가 꼭 확인, 주변 시세 대비 80% 이상이면 의심**

### 3. 대리인 사기
가짜 위임장으로 계약 → **집주인 본인에게 직접 확인 필수**

## 마무리: 계약 전 체크리스트 요약

1. 등기부등본 확인 (계약 당일 재확인)
2. 깡통전세 여부 계산
3. 전세보증보험 가입 가능 여부 확인
4. 집주인 신원과 세금 체납 확인
5. 이사 당일 전입신고 + 확정일자
6. 계약서에 특약 조항 기재
7. 안심전세 앱으로 추가 확인

이 7가지만 지키면 전세 사기의 90% 이상을 예방할 수 있어요. 귀찮더라도 꼼꼼히 확인하세요. 보증금은 여러분의 전 재산일 수 있으니까요.
